# Realm of Shadows - Text-Based RPG

A Python text-based RPG with:

- Player stats, leveling, and XP system
- Combat with random enemy encounters
- Critical hits, misses, and dynamic damage
- Loot system with inventory and equipable items
- Save/load game functionality

## How to Run

1. Install Python 3.11+
2. Run `pip install -r requirements.txt`
3. Run the game with:

```bash
python main.py
